# 📚 Library Management System

A full-featured web-based Library Management System built with HTML, CSS, and JavaScript. Designed to help libraries manage books, users, and transactions through a clean and intuitive interface — no backend server required.

📁 GitHub Repo: https://github.com/nazeeyabanuh/Library-Management-System_
Demo: https://library-git-main-nazeeyakulsumbanu6-5488s-projects.vercel.app/
---

## ✨ Features

- Book Management — Add, update, and delete books from the library catalogue
- Search Books — Instantly search by title, author, or category
- Book Categories and Languages — Organise books by genre/category and language
- Issue and Return Books — Track which books are issued, to whom, and when
- User Management — Add users and maintain their details and borrowing history
- Data Persistence — All data stored and managed via a local JSON file

---

## 🛠️ Tech Stack

- HTML5 — Structure and markup
- CSS3 — Styling and responsive layout
- JavaScript (Vanilla) — Logic, DOM manipulation, interactions
- JSON — Local data storage
